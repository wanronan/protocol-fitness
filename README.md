# Wanderson Performance — versão moderna

Redesign do projeto `wanronan/wan-performance`, mantendo os dados, treinos, dieta, metas e chaves de localStorage da versão anterior.

## Atualizar o GitHub

1. Abra o repositório `wanronan/wan-performance`.
2. Apague os arquivos antigos da raiz ou substitua-os.
3. Envie **todo o conteúdo desta pasta**, incluindo `css`, `js` e `assets`.
4. Faça o commit.
5. Aguarde o GitHub Pages atualizar.
6. No celular ou computador, faça uma atualização forçada ou limpe o cache do site se ainda aparecer a versão antiga.

## Compatibilidade com dados existentes

Mantidos os nomes usados anteriormente:

- `pesoAtual`
- `cinturaAtual`
- `treino_<dia>_<índice>`
- `dieta_<índice>`

Assim, os registros já salvos no mesmo navegador continuam disponíveis.
